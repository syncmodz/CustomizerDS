# CustomizerDS

![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)
![Platform: Nintendo 3DS](https://img.shields.io/badge/Platform-Nintendo%203DS-red)
![Language: C](https://img.shields.io/badge/Language-C-00599C)

Um homebrew elegante e fluido para Nintendo 3DS com customização completa:
- 🎨 **Fontes** (Comfortaa, MADE Evolve)
- 🌙 **Temas** (Claro/Escuro com transições yin-yang)
- 🎯 **Cores** (Picker HSV com preview)
- 💡 **LEDs RGB** (Modos RGB, Fixo, Pulso, Off)

Interface Glass-style (inspirada em Reva UI/cakeOS), 60fps reais, animações fluidas tipo Apple.

## Features
- ✨ 10+ transições de abertura de aba variadas (pool aleatório)
- 🎬 Animação yin-yang de tema com wipe radial
- 🔧 Touch & D-pad support (stylus, mouse, console controls)
- 🎪 Design tokens centralizados, modo claro/escuro completo
- 📊 Picker HSV interativo com seletor de cores predefinidas

## Build
```bash
cd CustomizerDS
make clean && make
# Gera: CustomizerDS.3dsx (emulador)
```

Abra em Citra ou Lime3DS.

## Controles
- **D-Pad Esq/Dir:** Navegar, ajustar valores
- **A:** Selecionar
- **B:** Voltar
- **START:** Confirmar
